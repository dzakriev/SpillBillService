# Сервис для разделения счета на компанию
Система автоматически распознаёт позиции из чека, распределяет их между участниками и рассчитывает сумму долга для каждого. Решает проблему ручного подсчёта и споров "кто сколько должен" после совместных посиделок.

## ДЗ 6: Инфраструктура как код. Terraform и Ansible

### Часть 1. Теоретическая: Выбор инструмента для задач

| № | Задача | Инструмент | Почему? |
|---|--------|------------|---------|
| 1 | Создать каталог (Folder) | Terraform | Создание облачного ресурса, управление жизненным циклом инфраструктуры |
| 2 | Создать VPC и подсеть | Terraform | Создание облачной сетевой инфраструктуры |
| 3 | Создать бакет в Object Storage | Terraform | Создание облачного ресурса хранения |
| 4 | Создать виртуальную машину | Terraform | Создание облачного вычислительного ресурса |
| 5 | Настроить группу безопасности | Terraform | Создание и настройка облачного ресурса безопасности |
| 6 | Установить python3, docker | Ansible | Конфигурация и установка ПО на существующей ВМ |
| 7 | Установить uv через pip | Ansible | Конфигурация и установка пакетов на существующей ВМ |
| 8 | Создать пользователя ml-dev | Ansible | Конфигурация системы на существующей ВМ |
| 9 | Скопировать requirements.txt | Ansible | Конфигурация и управление файлами на существующей ВМ |
| 10 | Удалить виртуальную машину | Terraform | Управление жизненным циклом облачного ресурса |

**Принцип разделения:**
- **Terraform** — для создания, изменения и удаления облачных ресурсов (инфраструктура)
- **Ansible** — для конфигурации и настройки уже существующих ресурсов (конфигурация)
